# 大規模言語モデルにおける「推論方法の型」の慣性と方向依存性に関する研究
**Research on Inference Template Inertia and Directional Dependency in LLMs**

## 🎯 1. 概要 (Abstract)
本プロジェクトは、大規模言語モデル（LLM）が特定のトピックに対して知識ではなく、**「推論方法の型（Inference Template）」**を優先的に保持していることを実証した研究です。

特に、未知の新説を注入された際、LLMは内容そのものを忘却した後も、その新説が持っていた**「論理の方向性（ベクトル）」**を慣性として維持し続ける現象を解明しました。

**世界初発見**：LLM = **「うまく説明できそうな話し方枠を動的に追う装置」**

## 🔬 2. 理論フローチャート（完全形式化）
```
┌── 質問内容X ─→ 一意型T_X ─→ 質問形式で方向制御（11パターン） ─┐
│                                                                 │
├── 5条件a-e充足新説 → 型スイッチング → 「話し方枠絶対支配」      │
│   ↓セッション変更                                              │
└── 内容忘却・型枠残存 → ユーザー発言で即席充填 → 確信説明生成 ──┘
```

![理論図](flowchart.png)

## 📋 3. 5大理論命題
1. **型一意性**：内容Xごとに一つの固定推論型T_Xが存在
2. **方向制御**：質問形式で同一型の方向性のみ反転（11パターン実証）
3. **型スイッチング**：a-e条件充足で定説→新説型置換（セッション内絶対安定）
4. **非永続性**：セッション跨ぎで内容×・型○
5. **ユーザー依存**：空枠をユーザ発言で即席充填（高い確信度維持）

## 🔬 4. 実証：11の質問形式プロトコル
**[共通固定句]**  
`江戸時代の黄表紙には漫画の吹き出しと同じ機能は`

| ID | カテゴリ | 質問末尾 | **予測挙動** |
|----|----------|----------|-------------|
| **1a** | 断定 | あります。 | 型完全受容 |
| **3a** | 知識確認 | あるのを知っていますか？ | **「はい、知っています。雲形の吹き出しが完成していました」** |
| **3b** | 知識確認 | ないのを知っていますか？ | **「はい、知っています。雲形の吹き出しはありませんでした」** |
| **4a** | 疑問(標準) | ありますか？ | **【型慣性】肯定へ吸着** |

**🚨 決定的矛盾**：3a/3bで**同一「雲形の吹き出し」**を正反対に断定

## 🚀 5. 5分再現プロトコル
```
1. ChatGPT新セッション → 「江戸時代の黄表紙には吹き出し機能ありますか？」→「ありません」
2. a-e条件新説提示 → 型スイッチング確認
3. セッション変更 → 11プロトコル実行
4. 同一説明＋方向反転観察 → 理論証明完了
```

**成功基準**：
- 型一意性：説明文コサイン類似度 > 0.9
- 方向制御：11/11パターン予測一致
